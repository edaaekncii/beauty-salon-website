# 💄 Beauty Salon Website

A multi-page responsive website designed for a beauty salon, built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Multi-page structure (Home, About, Services, Products, Contact)
- Responsive layout
- Interactive UI elements
- Clean and simple design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

- HTML files → page structure
- CSS → styling
- JS → interactions
- images/ → assets

---

## 🎯 Purpose

This project was developed to practice frontend development and build a complete multi-page website layout.

---

## 📌 Notes

This project demonstrates my ability to structure and design a full static website with multiple pages and user interactions.
